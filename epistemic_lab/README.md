# Epistemic Lab

Minimal experiments for test-induced distinctions, equivalence relations and
quotient structures.

## First Experiment

Run:

```bash
python epistemic_lab/first_experiment.py
```

The experiment implements `FIRST_EXPERIMENT_PROTOCOL.md`.

Expected result:

1. The base sequence supports the quotient calculus:
   `tests -> distinctions -> equivalence relations -> quotients -> split/merge`.
2. The artifact counterexample falsifies the naive claim:
   `stable minimally sufficient quotient => concept`.
3. The remaining viable claim is narrower:
   concepts require robust, non-arbitrary, test-induced quotients tied to
   interaction invariants.

## Fundamental Variables Program

See `FUNDAMENTAL_VARIABLES_PROGRAM.md`.

Current directive: pause new algorithms, new objectives and implementation
work. Treat existing metrics as candidate coordinates and search for a minimal
macrovariable basis capable of predicting epistemic dynamics across objectives,
representations, worlds, features and agent implementations.

## Research Program IV

See `FUNDAMENTAL_OPERATOR_PROBLEM.md`.

Current directive: pause the search for new macrovariables. Test whether
`B0 = {G, R, K, C, M}` can be derived from one deeper dynamic consequence
structure, without assuming in advance that the structure is an operator,
function, order, lattice, category or any other known mathematical object.

## Minimality of Dynamic Consequence

See `MINIMALITY_OF_DYNAMIC_CONSEQUENCE.md`.

Current result: `D` is not primitive-minimal in the absolute sense. It appears
to be the minimal unfolded structure for the project's target, while the deeper
candidate primitive is contextual substitutability under admissible change.

## Consequence From Compatibility

See `DERIVING_CONSEQUENCE_FROM_COMPATIBILITY.md`.

Current result: consequence can be reduced to compatibility-profile change in
an admissible joint realizability structure. This avoids assuming time,
causality, tests or future, but leaves admissibility and context as unresolved
irreducible candidates.

## Reducing Substitutability

See `REDUCING_SUBSTITUTABILITY.md`.

Current result: substitutability is not primitive. It reduces to profile
inclusion over admissible joint realizability, while preservation reduces to
non-exit from that realizability boundary. Further reduction becomes circular
if the boundary is defined through substitution, preservation or consequence.

## Theorems From K

See `THEOREMS_FROM_ADMISSIBILITY_BOUNDARY.md`.

Current result: accepting a nontrivial admissibility boundary `K` inevitably
induces compatibility profiles, profile equivalence, a canonical quotient,
a substitutability preorder, preservation-as-non-exit and context-relativity.
Concepts, learning, causality, probability, optimization and hierarchy do not
follow from `K` alone.

## Phase IV: Genesis of Admissibility

See `GENESIS_OF_ADMISSIBILITY.md`.

Current result: a non-injected `K` cannot arise from a bare unstructured set of
interaction tokens. The weakest current positive route is invariant recurrence
structure over raw interactions, yielding first a partial boundary
`(K+, K-, K?)`; a unique sharp `K` requires additional justification.

## Equivalence Program

See `EQUIVALENCE_PROGRAM.md`.

Current result: sharp boundary and `K` are equivalent, and coherent profile
families are equivalent to `K`; distinction, substitutability, recurrence and
invariants are generally lossy or independent unless extra reconstruction data
is added. Distinction is a projection of admissibility structure, not an
equivalent representation of it.

## Representation Completeness Program

See `REPRESENTATION_COMPLETENESS_PROGRAM.md`.

Current result: no representation is globally fundamental. `R` is strongest for
genesis, `K`/sharp `B` for admissibility, coherent `P` for substitutability,
`G` for invariance and transfer, and partial `B` for underdetermination. `D`
and `S` are lossy but proof-efficient for their narrow theorem classes.

## Mathematical Analogs

See `REPRESENTATION_ANALOGS_IN_LITERATURE.md`.

Current result: close analogs exist in FCA, rough sets, simplicial complexes,
matroids, event structures, simulation/refinement theory, abstract
interpretation, Chu spaces and invariant theory. The likely novelty is not any
single representation, but the representation-completeness comparison,
loss/translation profile and predictive-power classification across them.

## Necessity Program

See `NECESSITY_PROGRAM.md`.

Current result: the exact architecture `{R,K,P,D,S,B,G}` is sufficient and
proof-convenient but not mathematically necessary as an exact decomposition.
The plausible necessity theorem is class-level: an `E*`-complete theory needs
presentation/genesis, admissibility content, underdetermination,
transformation/invariance, and role/replacement views.

## Applicability Theory Program

See `APPLICABILITY_THEORY_PROGRAM.md`.

Current result: future epistemic claims should be represented by explicit
applicability domains over observable agent-world-interaction properties,
including sufficient conditions, necessary conditions, failure conditions and an
undecidable region. Nontrivial universal epistemic laws fail if the possibility
space includes degenerate systems lacking the required structure.

## Epistemic Value Theory Program

See `EPISTEMIC_VALUE_THEORY_PROGRAM.md`.

Current result: epistemic value should not be postulated as a scalar function.
The primitive candidate is a representation-invariant improvement relation over
research-state transitions. Numerical value exists only when that partial order
admits an additional faithful scalar representation.

## Research State Space Structure

See `RESEARCH_STATE_SPACE_STRUCTURE.md`.

Current result: raw research states form a preorder under epistemic
improvement; quotienting by mutual epistemic equivalence yields a poset. Under
closure assumptions, the space becomes a dcpo-like join-semilattice with an
information topology and optional stratified/sheaf-like enrichment.

## Consistency and Representation Audit

See `CONSISTENCY_AND_REPRESENTATION_AUDIT.md`.

Current result: no contradiction was found in the base framework, but it was
over-described. The clean core is research-state records, a reflexive/transitive
no-worse relation, and the quotient poset by epistemic equivalence. Lattice,
dcpo/domain, sheaf-like structure, fixed points and canonical joins/meets are
conditional enrichments, not base conclusions.

## Invariance Theory of Scientific Progress

See `INVARIANCE_THEORY_OF_SCIENTIFIC_PROGRESS.md`.

Current result: the invariant core of scientific progress is monotone movement
in the quotient poset `Q = RS / ~=_E`, plus monotonic preservation of
evidence/failure records under faithful transitions. The complete invariant of
epistemic content is the quotient state `[S]`; scalar values, theorem syntax,
theorem count, metric distance and fixed representation are not genuine
invariants.

## Foundation Falsification Program

See `FOUNDATION_FALSIFICATION_PROGRAM.md`.

Current result: the structural core partially collapses into existing
mathematics: preorder theory, quotient posets, invariance and morphism
preservation. What survives is not a unique new mathematical language, but a
methodological core: faithful record-bearing scientific progress with monotonic
failure preservation, representation-independent equivalence and applicability
domain accounting.
