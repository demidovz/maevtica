# Очередь опытов (готовы к запуску)

Каждый файл — готовый `args` для `cycle.workflow.js` (Workflow tool, scriptPath).
Запуск: `Workflow(scriptPath=..../cycle.workflow.js, args=<содержимое файла>)`.

| Файл | Что проверяет | Железо |
|---|---|---|
| `separability_boundary.json` | ⭐ ПОНИМАНИЕ: предсказать по внутренностям, где чистое отделение понятий держится, а где ломается — и **побить** ориентир Park (иначе «переоткрытие»). prior-art вшит в правило | CPU (стиринг) |
| `reflection_routing.json` | interp-подсказка «где рефлексировать» бьёт слепую рефлексию при равном бюджете? | CPU (проба) |
| `self_distill_lift.json` | ~~дистилляция в веса~~ — снята как исследовательская цель (переоткрытие STaR); держать только как навык | GPU |

Текущий фокус (по решению босса 2026-07-04): **понимание, не training-трюки.**
`separability_boundary` — прямое продолжение «съёмной категории», но нацелен на
слабость поля (граница провала separability), а не на его сильную сторону.
Порядок: `separability_boundary` → `reflection_routing`.
